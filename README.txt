SAO Website v2
==============
Dynamic responsive site with live category filtering.

1. Open index.html in a browser.
2. Products load from /data/products.json.
3. Replace image URLs with your own or local files in /assets/images/.
4. Fonts (RebellionKnight, Futura) go in /fonts/.
5. Edit colors or animations in css/style.css.
6. To add new products, just append to products.json with "category" and "image".
